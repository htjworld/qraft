# Qraft — Plan

Serverless React + TypeScript + Vite + WebGPU. GitHub Pages. No login, no cost.

## Styles (MVP)
| | Style | Dark module mapping |
|--|-------|---------------------|
| 🌸 | Cherry Blossom | Trunk / CherryBlossom / Grass |
| 🐱 | Cat | Nose / Fur / Ear / PawPad |
| 🐶 | Dog | Nose / Fur / FloppyEar / Bone |

## User Flow
Visit → random style → type URL → live QR → tap dots button to switch style → tap canvas to toggle 3D ↔ 2D

## Layout
- Canvas fills screen (mobile-first)
- Floating dots button top-right → emoji-only popover (🌸 🐱 🐶)
- URL input pinned bottom, responsive

## Out of Scope (post-MVP)
Canvas 2D fallback · PNG export · custom colors · PWA
