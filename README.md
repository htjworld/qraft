# Qraft

<a href="https://htjworld.github.io/qraft">🔗 Service URL</a>

> URL을 입력하면 나무 형태의 3D QR코드로 변환해주는 WebGPU 기반 QR 메이커

## Features

- 실시간 렌더링 — URL 입력 즉시 3D QR코드 생성, 로딩 없음
- 4가지 나무 테마 — 봄나무, 벚꽃나무, 가을나무, 겨울나무 중 선택
- 3D ↔ 2D 전환 — 캔버스 클릭으로 스캔 가능한 평면 QR로 전환
- 서버리스 — 서버 없음, 회원가입 없음, 무료
